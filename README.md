
```shell
npm run serve
```

### Compiles and minifies for production

``` shell
npm run build
```

### Lints and fixes files

``` shell
npm run lint
```

### 目录

```txt
src为主目录
components文件夹为全局通用组件
less文件夹为全局通用css及配置
router为路由配置
store为vuex配置
views为具体页面
views下各页面文件夹代表各页面
各页面文件夹中components文件夹为该类页面专用组件
```
